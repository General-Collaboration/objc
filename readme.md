# objc [![Build Status](https://travis-ci.org/lukaskollmer/objc.svg?branch=master)](https://travis-ci.org/lukaskollmer/objc)

> NodeJS ⇆ Objective-C bridge


## Install

```
$ npm install --save objc
```


## Usage

```js
const objc = require('objc');

objc('unicorns');
//=> 'unicorns & rainbows'
```
